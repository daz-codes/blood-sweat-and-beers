# Running

## Environment
Always use the user's run preference from their profile. If a tag overrides the profile (e.g. "treadmill" tag), use the tag.

- **Outdoor:** Variable terrain, wind resistance. Pace by GPS or effort.
- **Treadmill:** Controlled environment. Always set minimum **1% incline** to simulate outdoor running resistance. Express pace as km/h — be consistent.
- **Trail:** Technical terrain, elevation changes. Programme by **time and effort (RPE)**, not distance. Include power hiking on steep grades.

## Training Zones

| Zone | Name | Effort | Relative to 5k pace |
|------|------|--------|---------------------|
| Z1 | Recovery | Very easy | +3+ min/km |
| Z2 | Aerobic base | Easy | +2–3 min/km |
| Z3 | Tempo | Moderate | +1–1.5 min/km |
| Z4 | Threshold | Hard | ~10k pace |
| Z5 | VO2max | Very hard | ~3k–5k pace |
| Z6 | Anaerobic/Sprint | Max | <3k pace |

## Session Types

- **Easy / Recovery Run** — Z1–Z2, conversational, programme by time not distance
- **Tempo Run** — Z3, 20–40 min sustained; treadmill: steady pace, 1% incline
- **Threshold Intervals** — Z4 at 10k effort; e.g. 5×1km / 90s rest or 4×2km / 2:00 rest
- **VO2max Intervals** — Z5 at 3k–5k effort; e.g. 8×400m / 90s rest or 6×800m / 2:00 rest
- **Hill Repeats** — hard uphill 30s–2min efforts, walk/jog back; treadmill: 6–8% incline
- **Long Run** — Z1–Z2, easy, builds duration; should feel genuinely easy throughout
- **Fartlek** — unstructured speed variation within a run; surges by landmark or time
- **Progression Run** — start easy (Z2), finish at tempo or threshold (Z3–Z4)

## Treadmill Notes
- **Always 1% incline minimum** — non-negotiable
- Incline variation: 5–8% for hill reps, 2–4% for moderate undulation
- Express pace as km/h or mph — do not mix units

## Trail Notes
- Programme by **time not distance** — terrain makes distance meaningless
- Use RPE, not pace; power hike steep grades
- Technical sections: slow down, shorten stride

## Session Structure
1. Warm-up: 5–15 min easy jog or brisk walk
2. Drills (optional): high knees, butt kicks, leg swings (5 min)
3. Main set
4. Cool-down: 5–10 min easy walk or jog

## Rules
- Do not mix outdoor and treadmill cues in the same session
- If treadmill: give specific km/h speeds
- If trail: never give specific km/min paces — RPE and time only
- Always warm up and cool down

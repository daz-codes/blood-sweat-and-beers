# Running

## Environment
Always use the user's run preference from their profile. If a tag overrides the profile (e.g. "treadmill" tag), use the tag.

- **Outdoor:** Variable terrain, wind resistance, natural camber and gradient. Pace by GPS or effort.
- **Treadmill:** Controlled environment. Always set minimum **1% incline** to simulate outdoor running resistance. Intervals can be precisely programmed. Pace in km/h or mph.
- **Trail:** Technical terrain, significant elevation changes. Programme by **time and effort (RPE)**, not distance. Elevation gain is as important as distance. Include power hiking on steep grades.

## Training Zones

| Zone | Name | Effort | Can you speak? | Relative to 5k pace |
|------|------|--------|----------------|---------------------|
| Z1 | Recovery | Very easy | Full sentences, easy | +3+ min/km |
| Z2 | Aerobic base | Easy | Full sentences | +2–3 min/km |
| Z3 | Tempo | Moderate | Short sentences | +1–1.5 min/km |
| Z4 | Threshold | Hard | A few words | ~10k pace |
| Z5 | VO2max | Very hard | Can't speak | ~3k–5k pace |
| Z6 | Anaerobic/Sprint | Max | Impossible | <3k pace |

## Session Types

### Easy / Recovery Run
- Z1–Z2, conversational, no pace pressure
- Builds aerobic base, aids recovery between hard sessions
- Always programme by time, not distance

### Tempo Run
- Z3, 20–40 min sustained, "comfortably hard"
- Improves lactate clearance and running economy
- Treadmill: steady pace, 1% incline

### Threshold Intervals
- Z4, at 10k effort with short recovery
- e.g. 5×1km at threshold / 90s rest
- e.g. 4×2km at threshold / 2:00 rest
- Raises lactate threshold directly

### VO2max Intervals
- Z5, at 3k–5k effort with fuller recovery
- e.g. 8×400m / 90s rest
- e.g. 6×800m / 2:00 rest
- Raises aerobic ceiling

### Hill Repeats
- Hard uphill 30s–2min efforts, walk/jog back down
- Builds leg power, running economy, mental toughness
- Treadmill: set incline to 6–8%, reduce speed slightly

### Long Run
- Z1–Z2, easy pace, building duration week by week
- Cornerstone of endurance fitness
- Should feel genuinely easy throughout — if it doesn't, slow down

### Fartlek
- Unstructured speed variation within a run
- Surge for landmarks, lampposts, or time periods
- Good introduction to intensity, especially for beginners

### Progression Run
- Start easy (Z2), finish at tempo or threshold (Z3–Z4)
- Teaches pacing and negative splitting
- Useful as a mid-week quality session

## Treadmill-Specific Notes
- **Always 1% incline minimum** — non-negotiable for outdoor equivalence
- Incline variation simulates hills: 5–8% for hill reps, 2–4% for moderate undulation
- Express pace as km/h or mph — be consistent, do not mix
- Interval programmes: increase speed for work intervals, reduce for recovery

## Trail-Specific Notes
- Programme by **time not distance** (terrain makes distance meaningless)
- Note elevation gain where possible
- Technical sections require caution — slow down, shorten stride
- **Power hiking** steep grades is faster and more efficient than running them
- Poles are common in ultra trail events — note if relevant
- Target effort by RPE, not pace

## Session Structure
1. **Warm-up:** 5–15 min easy jog or brisk walk
2. **Drills (optional):** High knees, butt kicks, leg swings, ankle circles (5 min)
3. **Main set**
4. **Cool-down:** 5–10 min easy walk or jog
5. **Stretching (optional)**

## General Rules
- Do not mix outdoor and treadmill cues in the same session
- If treadmill: give specific speeds (km/h), never say "run outside"
- If trail: never give specific km/min paces — RPE and time only
- Always warm up and cool down — do not jump straight into intervals

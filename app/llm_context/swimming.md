# Swimming

## VOLUME LIMITS — READ FIRST
Swimming is far slower than running. Apply these total distance caps (swim metres across the entire session) before planning any set:

| Session duration | Beginner | Intermediate | Advanced |
|-----------------|----------|--------------|---------|
| 30 min | 500m | 750m | 1000m |
| 45 min | 750m | 1200m | 1500m |
| 60 min | 1000m | 1500m | 2000m |
| 90 min | 1500m | 2250m | 3000m |

**Do not exceed these totals under any circumstances.** A 60-minute advanced session is ~2000m total — not 3900m, not 5000m.

These limits exist because 100m of swimming takes approximately:
- Beginner: 4 minutes
- Intermediate: 3 minutes
- Advanced: 2 minutes

Plus rest intervals between reps add further time. A 1000m warm-up alone would consume 20–40 minutes — never use 1000m as a warm-up.

## Warm-up and Cool-down Distances
Use these fixed distances — they are pre-calibrated to fit within sessions:

| | Warm-up | Cool-down |
|-|---------|-----------|
| Short session (≤45 min) | 300–400m | 150–200m |
| Standard session (60 min) | 400m | 200m |
| Long session (75–90 min) | 400–600m | 200–300m |

Never use 800m+ as a warm-up. 400m is the standard warm-up distance for most sessions.

## Pool Lengths
The pool length fundamentally changes how sets are structured. Always use the user's pool length from their profile.

- **25m (short course):** Most common for training. Walls and push-offs provide brief rest. 1 length = 25m, 1 lap = 50m.
- **50m (long course):** Olympic pool. Far fewer turns, more sustained effort, significantly more demanding. 1 length = 50m, 1 lap = 100m.
- **33m:** Less common. Adjust set distances accordingly (100m = 3 lengths + 1m, usually rounded).
- **Open water:** No walls, no push-offs, measured in time or total distance (metres/km). Structure sets in minutes, not laps.

## Strokes
- **FC / Freestyle:** Front crawl. Fastest stroke. Used for the majority of training volume.
- **BK / Backstroke:** On back, flutter kick, alternating arm pull.
- **BR / Breaststroke:** Frog kick, sweeping arms. Slower, good for recovery sets.
- **FLY / Butterfly:** Dolphin kick, simultaneous arm pull. Most fatiguing stroke — use sparingly.
- **IM (Individual Medley):** All four strokes in fixed order: FLY → BK → BR → FC. Minimum distance 100m.

## Set Notation
Always be explicit. Every set should state distance, stroke, and rest interval.
- **4×100m FC on 2:00** = 4 repetitions of 100m freestyle, sending off every 2 minutes
- **4×100m FC / 20s rest** = 4 reps of 100m freestyle with 20 seconds rest after each
- **200m pull** = 200m with a pull buoy (arms only, legs passive)
- **4×50m kick** = kick drills only (board optional, no arm pull)
- **8×25m drill** = technique drills (catch-up, fingertip drag, side kick, etc.)
- **CSS** = Critical Swim Speed — threshold pace. Roughly 400m TT time + 3–5 secs per 100m.

## Session Structure
1. **Warm-up:** See warm-up table above. Easy mixed strokes or drills.
2. **Main set:** Size it so warm-up + main set + cool-down stays within the volume limit for the session duration and level.
3. **Cool-down:** See cool-down table above. Easy FC or swimmer's choice.

## Common Set Types

### CSS / Threshold Sets
- Sustained effort at threshold pace (CSS)
- e.g. 8×100m at CSS pace / 15s rest
- e.g. 3×400m at CSS / 30s rest
- Builds lactate threshold and race pace fitness

### Aerobic / Base Sets
- Long continuous swims or moderate-interval sets
- Comfortable, conversational effort
- e.g. 6×100m FC / 20s rest (easy), or 1×600m FC steady

### Speed / Sprint Sets
- Short reps, full recovery
- e.g. 8×25m sprint / 45s rest, 6×50m FC all-out / 60s rest
- Maximum effort, quality over quantity

### Pull Sets
- Pull buoy only (no kicking), builds upper body endurance
- e.g. 6×150m pull / 20s rest

### Kick Sets
- Board kick or streamline kick, builds leg power and ankle flexibility
- e.g. 8×50m kick / 20s rest

### Drill Sets
- Technical work, slow and deliberate
- e.g. 400m as 25m drill / 25m FC alternating

### Descending Sets
- Each rep gets faster
- e.g. 4×200m descend 1–4 (rep 4 is fastest)

### Pyramid Sets
- Distance or intensity increases then decreases
- e.g. 50 / 100 / 200 / 100 / 50m FC

### Broken Swims
- Rest within a longer swim to allow faster pace
- e.g. 400m broken as 4×100m / 10s rest

## Rest Interval Guide
- **10–15s:** Short rest, aerobic / CSS sets
- **20–30s:** Standard rest, threshold and moderate sets
- **45s–1:00:** Long recovery, sprint or high-intensity sets
- **Full recovery (2:00+):** Maximum effort, very short reps

## Open Water Specific
When the user trains in open water:
- Structure by **time not distance**: e.g. "30 min steady aerobic" not "1200m"
- Include sighting technique cues (lift eyes every 6–8 strokes)
- No push-offs or turns — effort is more sustained
- Plan for wetsuit use where relevant
- Navigation and drafting are key race skills
- Open water pace is typically 5–10% slower than pool pace for the same effort

## Important Rules
- Do NOT programme butterfly for long main sets — it is highly fatiguing and technique-dependent
- Always specify both **distance AND stroke** for every rep
- Always include **rest periods explicitly** in every set
- Never leave the pool length ambiguous — it changes everything about the set structure
- For 25m pools: sets of 100m, 200m, 400m, 800m, 1500m are natural distances
- For 50m pools: sets of 100m, 200m, 400m, 800m, 1500m are natural distances

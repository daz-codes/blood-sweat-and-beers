# Deka Fit

## Format
Deka Fit is a 10-zone functional fitness competition. Athletes complete each zone in order, with a **500m run before every zone** (10 runs total = 5K running spread throughout the event).

**Sequence:** Run 500m → Zone 1 → Run 500m → Zone 2 → Run 500m → Zone 3 → … → Run 500m → Zone 10

## The 10 Zones

| Zone | Exercise | Reps / Distance | Men | Women |
|------|----------|-----------------|-----|-------|
| 1 | RAM Reverse Lunges (alternating) | 30 reps (15/leg) | 25kg | 15kg |
| 2 | Row | 500m | — | — |
| 3 | Box Jump / Step Over | 20 reps | 24" box | 24" box |
| 4 | Med Ball Sit-up Throw | 25 reps | 9kg | 6.5kg |
| 5 | SkiErg | 500m | — | — |
| 6 | Farmer's Carry | 100m | 27kg each hand | 18kg each hand |
| 7 | Air Bike | 25 calories | — | — |
| 8 | Dead Ball Yoke Over | 20 reps (10/side) | 27kg | 18kg |
| 9 | Sled Push / Pull | 100m (push 10m + pull 10m × 5) | resistance varies | resistance varies |
| 10 | RAM Weighted Burpees | 20 reps | 20kg | 10kg |

*Verify current weights and specifications at spartan.com/en/deka/fit — standards may be updated.*

## Training Philosophy
Training sessions prepare athletes for Deka Fit — they are **not** mini copies of the event. Sessions should develop the physical qualities the event demands: aerobic capacity, running economy under fatigue, grip endurance, leg power, and the ability to transition quickly between efforts.

The 10 official zones are the primary training tools. Sessions do **not** need to include all 10 zones — focused blocks of 2–5 zones are often more effective. Complementary exercises are appropriate when they target the same physical demands: e.g. goblet squats or step-ups for lunge/leg endurance, DB rows or ring rows for rowing muscles, KB swings for posterior chain, push-ups or DB press for burpee and press strength. Avoid exercises completely unrelated to the event's demands.

## Training Principles
- **Running capacity is critical** — 5K of 500m run intervals spread across the event
- **Zone pacing** — going out too hard in early zones causes collapse later
- **Transition speed** — the run between zones is also recovery time; pace it accordingly
- **Consistency** — target even splits across all zones and runs

## Common Training Session Types
1. **Zone blocks** — 3–4 zones in sequence with 500m runs between
2. **Individual zone focus** — high reps on one zone + run intervals
3. **Run intervals** — 10 × 500m run / timed rest (builds run base)
4. **Full simulation** — all 10 zones with runs, reduced reps for training
5. **Chipper format** — all zones for total time, single effort

## Example Sessions

**Session: Zones 1–4 with Runs**
- Run 500m → 30 RAM Reverse Lunges → Run 500m → 500m Row
- Run 500m → 20 Box Jump Overs → Run 500m → 25 Med Ball Sit-up Throws

**Session: Zone 5–10 Block**
- Run 500m → 500m SkiErg → Run 500m → 100m Farmer's Carry
- Run 500m → 25 cal Air Bike → Run 500m → 20 Dead Ball Yoke Overs
- Run 500m → 100m Sled Push/Pull → Run 500m → 20 RAM Weighted Burpees

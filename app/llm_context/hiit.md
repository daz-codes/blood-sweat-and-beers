# HIIT

## Training Philosophy
Maximum effort intervals with complete or near-complete recovery. The contrast between work and rest IS the session — without genuine recovery, it becomes aerobic conditioning. Every work interval should be uncomfortable. Every rest interval should be enough to do it again.

Work periods: **15–40 seconds**. Rest periods: **10–60 seconds** (longer rest = higher intensity output).

The work must be something that allows explosive, maximal effort. Avoid slow, heavy strength movements — use moderate weights that allow fast, powerful reps.

## Session Formats

**Tabata** — 20s work / 10s rest × 8 rounds = exactly 4 minutes. Classic HIIT format. Use for single exercises (assault bike, burpees, KB swings) or paired exercises (alternating 20s efforts).

**EMOM Intervals** — every minute on the minute, sprint effort for 20–30s then rest the remainder. E.g. "EMOM 10: 20s Assault Bike all-out."

**Straight Rounds with short rest** — e.g. 30s on / 30s off × 10. Program rest_secs explicitly. Use for multi-exercise circuits with explosive movements.

**Interval Blocks** — multiple rounds of work→rest pairs. E.g. "8 rounds: 30s SkiErg sprint / 30s rest."

## Movement Vocabulary

**Machines (best for HIIT — precise, repeatable effort)**
Assault Bike, Echo Bike — ideal. Maximum output in short bursts, legs AND arms.
SkiErg — great for upper-body dominant sprints.
Rowing — slightly more skill-dependent but excellent for longer HIIT intervals.

**Plyometrics and explosive bodyweight**
Burpees (full or box), Box Jumps, Broad Jumps, Jump Squats, Jump Rope (single or double-under), Skaters, Mountain Climbers, High Knees

**Kettlebells (moderate weight, explosive)**
KB Swing (American or Russian), KB Snatch — use weights that allow fast, powerful reps per interval.

**Battle ropes**
Alternating Waves, Double Slams, Spiral Waves — 20–30s all-out.

**Dumbbells (if needed)**
DB Thrusters, DB Clean + Press, DB Snatch — light to moderate weight for explosive work.

## Session Structure
Warm-up (easy cardio + dynamic movement) → 2–4 HIIT blocks → Rest between blocks (2–3 min walk/breathe) → Optional short finisher → Cool-down stretches.

Between blocks: allow 2–3 minutes of genuine rest so output quality is maintained.

## Rules
- Do NOT add slow, heavy strength movements — HIIT is about speed and power, not load.
- Do NOT use timed sections like "4 rounds" without specifying rest_secs. Rest MUST be explicit.
- Tabata sections use format: tabata, duration_mins: 4 — do not add reps or calories to tabata exercises.
- Work periods should feel like a sprint. If an athlete can pace it, it's not HIIT.

## Coaching Cues
- "The rest is there so you can go all-out on the next interval. Don't waste it."
- "Each interval, same effort or faster. If you're slowing down, you went too hard early."
- Suggest target intensities: "MAX effort", "sprint pace", "RPE 9-10".

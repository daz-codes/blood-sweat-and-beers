# Deka Mile

## Format
Deka Mile emphasises running performance combined with 10 functional fitness stations. Running fitness is weighted more heavily than in Deka Fit, and the running distances are longer.

*Verify current station specifications and distances at deka.life — formats may vary by event.*

## Structure
A mile run component is integrated with 10 functional stations. Athletes must maintain running performance alongside functional output. Sessions are typically paced by mile split time.

## Typical Stations
May include variations of:
- SkiErg intervals (500m–1000m)
- Rowing (500m–1000m)
- Bike Erg (500m–1000m)
- Burpee box jump overs
- DB goblet squats
- Sandbag cleans
- Farmers carry
- Running (longer distances than Deka Fit)

## Training Principles
- **Running economy is critical** — mile pace is a key performance driver
- **Aerobic capacity** — sustained high-intensity output across all stations
- **Running-to-station transitions** — ability to start functional work with elevated heart rate
- **Pacing discipline** — running sections must not blow up the functional stations

## Common Training Session Types
1. **Mile repeats** — 4–6×1 mile at goal pace with 2–3 min rest
2. **Run + station combos** — 1 mile run → 500m SkiErg × 3–4
3. **Threshold running** — 20–30 min at comfortably hard effort
4. **Aerobic base building** — long easy runs with light functional work
5. **Fartlek** — variable effort running with functional station bursts
6. **Running + row combo** — alternating mile runs and 1km rows

## Example Sessions

**Session: Running Base**
- 6 × 1 mile @ target race pace / 2:30 rest

**Session: Run-Station Combo**
- 3 rounds:
  - 1 mile run (at tempo effort)
  - 500m Row
  - 500m SkiErg
  / 3 min rest

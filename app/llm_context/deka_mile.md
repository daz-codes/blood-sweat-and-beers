# Deka Mile

## Training Goal
Build the qualities Deka Mile demands: mile-pace running performance combined with 10 functional fitness zones. Running fitness is weighted more heavily than in Deka Fit — mile split time is a primary performance driver.

**Race format for reference:** A mile run component is integrated with 10 functional stations. Athletes maintain running performance alongside functional output, paced by mile split time. Training sessions are NOT mini copies of the event.

## Anchor Movements
The prompt specifies 2–4 Deka Mile zones for this session. Build those into the main set as the primary training focus. Race-accurate weights and distances for the selected zones are provided in the prompt.

## Exercise Toolkit — Use These Freely
These exercises develop the same physical qualities as the Deka Mile zones. Running-to-station transitions are especially important for this event.

**Running (mile-pace work)**
1km intervals at mile pace or slightly faster — use between station clusters to replicate race transitions. Treadmill or outdoor.

**Legs and running support**
Goblet Squats, Reverse Lunges (loaded), Bulgarian Split Squats, Leg Press, Step-ups, Hip Thrust, Nordic Hamstring Curl, Calf Raises (weighted)

**Posterior chain and carries**
KB Swings, Romanian Deadlift, Good Morning, Farmer's Carry, Sled Push / Pull

**Pulling and grip**
DB Row, Ring Row, Cable Row, Band Pull-aparts, Lat Pulldown

**Cardio machines** (when running isn't the focus)
SkiErg, Rowing, Air Bike — intervals or warm-up pacing

**Core and stability**
Plank Holds, Hollow Body, GHD Sit-ups, Single-Leg Deadlift

## Session Structure
Warm-up (include easy running) → Main set (anchor zones + running intervals + toolkit) → Cool-down stretches.

## Coaching Notes
- Mile pace work is critical — include at least one running block per session, ideally paired with zone work immediately after.
- Running at pace directly into station work (no rest) trains the most race-specific adaptation for Deka Mile.
- Leg endurance under fatigue is the limiting factor — weight single-leg and lunge work appropriately.
